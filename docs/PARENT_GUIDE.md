# 保護者ガイド

## 保護者画面

`/parent` から保護者画面を開きます。初回は4桁PINを設定します。

## PINについて

PINは子どもの誤操作を防ぐための簡易機能です。PIN本体は保存せず、saltとhashのみ保存します。強固な認証やクラウド保護を目的としたものではありません。

## 見られる情報

学習日数、総回答数、正解数、正答率、学習時間、最近の学習日、習得文字数、苦手文字数、復習予定、回答時間の変化、日別履歴を確認できます。

## 設定

保護者画面では、アニメーション軽減、文字サイズ、標準問題数、PIN変更、PIN解除、バックアップ、復元、データ初期化を扱います。

## データ初期化

初期化にはPINと確認文字「リセット」が必要です。実行前にバックアップを作ることをおすすめします。
