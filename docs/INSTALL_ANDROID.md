# Androidでのインストール

## 公開後の手順

1. 公開URLをChromeで開きます。
2. 右上のメニューを開きます。
3. 「ホーム画面に追加」または「アプリをインストール」を選びます。
4. ホーム画面にアイコンが出たことを確認します。
5. アイコンから起動します。
6. 一度オンラインで開きます。
7. 機内モードなどでオフライン確認をします。
8. データはブラウザ内に保存されます。
9. Chromeのデータ削除で消える可能性があります。
10. 定期的にバックアップしてください。

## URL差し替え欄

公開URL：`https://example.com/moji-bouken/`

公開後に、このURLを実際のURLへ差し替えてください。
