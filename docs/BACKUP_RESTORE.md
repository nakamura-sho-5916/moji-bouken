# バックアップと復元

## バックアップ

1. 保護者画面を開きます。
2. PINを入力します。
3. 「バックアップ」を開きます。
4. 「バックアップ出力」を押します。
5. JSONファイルを安全な場所に保管します。

バックアップには、学習データ、世界の進み具合、もちもの、図鑑、設定が含まれます。PIN本体や一時的なセッション情報は含まれません。

## 復元

1. 保護者画面を開きます。
2. 「バックアップ」を開きます。
3. JSONファイルを選びます。
4. 表示された内容を確認します。
5. 復元すると、いまのデータは選んだバックアップに入れ替わります。

復元前には、現在のデータも自動でバックアップされます。
