# 困ったとき

## アプリが開かない

一度オンラインで開いてから、もう一度試してください。開発中は `npm run dev` または `npm run preview` が起動しているか確認します。

## データが見つからない

ブラウザのデータ削除や別ブラウザの利用で、保存先が変わることがあります。バックアップファイルがあれば復元できます。

## PINを忘れた

MVPではPIN再発行機能はありません。バックアップがある場合は、復元でPINなしの状態に戻せます。

## オフラインで動かない

先にオンラインで一度開き、Service Workerに必要なファイルを保存させてください。

## 復元できない

ファイル形式、バージョン、ファイルサイズ、JSONの内容を確認してください。不正なファイルは受け付けません。
